[h1] What does this mod do? [/h1]
This mod is a small addition to [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1293725945] Augmentations [/url] that uses a newer hook in the CHL to change the status display for units in need of an Augment.
The status text should also tell you what body part needs an augment.

It also has some new inventory images for all tiers of augmentations.

[h1] Config Options [/h1]
Options in the xcomgame.ini config are to enable/disable the inventory images and what colour the text appears as.
By default I selected a faded cyan, as this looks both metallic to me and like the soldier is disabled.
